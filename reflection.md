# Exercise 1 Reflection

In Exercise 1, I created a GitHub repository called my-essay and made three separate commits for my essay. The first commit added the title, the second commit added the introduction, and the third commit added the body paragraph.

I used the command line to check my history with `git log --oneline` and to compare changes with `git diff HEAD~2 HEAD`.

The GitHub website helped me understand the changes better because it was more visual. I could click each commit and see exactly what was added each time.

Yes, I can see how my essay developed over time because each commit shows one step in the writing process.
